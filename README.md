# RetoNewsPaperAPI

RetoNewsPaperAPI is a simple iOS app built with SwiftUI that fetches science news from the Spaceflight News API and presents a list with a detail view.

## Features
- Browse recent science articles
- Read article details with images
- Handles loading and basic error states

## Tech Stack
- SwiftUI
- Combine
- URLSession
- XCTest

## Getting Started
1. Open `RetoNewsPaperAPI.xcodeproj` in Xcode.
2. Select a simulator or device.
3. Run the app (Cmd + R).

## Tests
- Run tests in Xcode: Product > Test (Cmd + U)

## Data Source
- Spaceflight News API: https://api.spaceflightnewsapi.net/
